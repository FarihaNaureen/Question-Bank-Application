# Question Bank Application

A Spring Boot MVC based Question Bank Application developed using Java, Spring Boot, Thymeleaf, MySQL, and Maven.

## Features
- Admin login
- Add and manage questions
- Coding question support
- User authentication
- Dashboard management
- Database integration

## Technologies Used
- Java
- Spring Boot
- Spring MVC
- Thymeleaf
- MySQL
- Maven
- HTML/CSS

## How to Run
1. Clone the repository
2. Open in Spring Tools Suite / Eclipse
3. Configure MySQL database
4. Run the Spring Boot application
